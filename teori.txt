Bot WhatsApp merupakan teknologi yang baru baru ini ada semenjak framework javascript bernama baileys atau @whiskeysockets/baileys.
framework ini merupakan apikey whatsapp yang terhubung menggunakan server whatsapp, meskipun framework ini bukan resmi dari WhatsApp namun 
gunakan dengan bijak, dan framework ini juga open source dan memiliki library code atau sumber kodenya bisa didapat di situs github atau situs npm (node package manager).
framework ini berguna jika aplikasi WhatsApp kalian digunakan layanan sosial mapaun layanan masyarakat terbuka.
